# SkyBound v2

Модульный SkyBlock плагин с архитектурой **Core + Addon** для Minecraft серверов на базе Spigot/Paper 1.16.5+.

## Описание

SkyBound v2 — это полнофункциональное ядро для SkyBlock серверов, предоставляющее все необходимые механики: острова, экономику, команды, миссии, магазин, улучшения, бустеры, лидерборд и многое другое. Плагин поддерживает систему аддонов для расширения функционала.

## Архитектура

```
skybound-api    → Публичные интерфейсы, модели, события (jar для всех аддонов)
skybound-core   → Основной плагин: острова, экономика, команды, улучшения, генераторы, миссии, магазин, банк, бустеры, лидерборд
```

## Возможности

| Функция | Описание |
|---------|----------|
| **Острова** | Создание, удаление, регенерация островов с поддержкой Overworld, Nether, End |
| **Банк** | Общий командный банк с валютой и кристаллами |
| **Экономика** | Интеграция с Vault, внутренняя валюта |
| **Генераторы** | Многоуровневый генератор руд с настраиваемым распределением |
| **Миссии** | Многоуровневые миссии с авто-отслеживанием (добыча, размещение, убийства, крафт, рыбалка и др.) |
| **Улучшения** | Постоянные улучшения острова (размер, команда, генератор, хопперы, посевы, спавнеры) |
| **Бустеры** | Временные баффы острова (фарм, XP, спавнеры, генератор, полёт) |
| **Магазин** | Категорийный магазин покупки/продажи с командными предметами |
| **Команды (Teams)** | Приглашение, кик, повышение, понижение, доверие, ко-оп, передача владения |
| **Лидерборд** | Топ островов по стоимости или уровню |
| **Сезоны** | Сезонная система с наградами и сбросами |
| **Ежедневные/Еженедельные квесты** | Регулярные задания с наградами |
| **Торговля** | Система обмена между игроками |
| **Посещения и лайки** | Посещение чужих островов, система лайков |
| **Варпы** | Игроки устанавливают варпы, видимые посетителям |
| **Кастомные рецепты** | Настраиваемые рецепты крафта |

## Команды

| Команда | Описание |
|---------|----------|
| `/is create` | Создать остров |
| `/is home` | Телепортироваться на свой остров |
| `/is members` | Управление участниками острова |
| `/is settings` | Настройки острова |
| `/is trade` | Торговля с другим игроком |
| `/is like` | Поставить лайк острову |
| `/is warps` | Список публичных варпов |
| `/is warp <name>` | Телепортироваться на варп |
| `/is setwarp <name>` | Установить варп на острове |
| `/is chest` | Общее хранилище острова |
| `/skyadmin` | Административные команды |

## Система аддонов

SkyBound v2 поддерживает расширение через аддоны. Каждый аддон реализует интерфейс `SkyBoundAddon`:

```java
public interface SkyBoundAddon {
    void onEnable(SkyBoundAPI api);
    void onDisable();
    String getName();
}
```

Аддоны могут работать как в связке с SkyBound, так и автономно.

## API

Доступ к сервисам через реестр `SkyBoundAPI`:

```java
SkyBoundAPI api = SkyBoundAPI.get();
IslandProvider islands = api.getIslandProvider();
EconomyProvider economy = api.getEconomyProvider();
TeamProvider teams = api.getTeamProvider();
```

### События (Events)

- `IslandCreateEvent`, `IslandDeleteEvent`
- `IslandMemberJoinEvent`, `IslandMemberLeaveEvent`
- `IslandLevelUpEvent`, `IslandUpgradeEvent`
- `GeneratorProduceEvent`, `MissionCompleteEvent`
